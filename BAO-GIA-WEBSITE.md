## 2. CÔNG NGHỆ SỬ DỤNG

- **Frontend:** Astro + Tailwind CSS
- **Ngôn ngữ:** TypeScript / JavaScript
- **Hiệu ứng:** GSAP (GreenSock) + ScrollTrigger
- **Triển khai (Deploy):** Firebase Hosting (CDN toàn cầu, HTTPS, tốc độ cao)
- **Đặc điểm:** Website tĩnh hiệu năng cao, tải nhanh, chuẩn SEO, responsive đầy đủ trên điện thoại – máy tính bảng – máy tính.
